### 2.3 记忆架构的技术演进

#### 三层存储架构（主流方案）

```
用户记忆层（User Memory）
    ↓

    ↓
Agent记忆层（Agent Memory）
```

**存储后端**：混合向量 + 图数据库 + 键值存储

- **向量数据库**：语义相似性检索

- **键值存储**：快速精确访问

- **run_id**：单次任务会话
- **app_id**：应用上下文
