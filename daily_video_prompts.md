### 《时光不在》- 治愈系等待主题

```
治愈系电影风格，日式动漫质感，柔光朦胧感，温馨治愈的情绪基调。

0-5秒：午后的咖啡馆窗边，阳光透过玻璃在桌面投下温暖的光斑，一杯刚泡好的卡布奇诺冒着热气，杯子上映着窗外蓝天的倒影。女孩坐在窗边，手里翻着一本书，嘴角带着浅笑。

5-10秒：窗外的光线渐渐从正午的白光变成黄昏的橘色，咖啡杯已经空了，杯底留着一圈咖啡渍。女孩抬起头看向门口，眼神里有一丝期待，但很快又低下頭，继续看书，动作变得缓慢。背景音乐渐弱，只剩雨声和若有似无的钢琴余韵。

10-15秒：窗外已经完全入夜，街灯亮起，桌上的咖啡杯旁多了一本翻到中间的书和一支干枯的花。女孩轻轻笑了笑，站起身离开，背影在暖黄色灯光下显得释然。镜头缓缓拉远，咖啡馆重新陷入宁静。画面定格在"有些人注定只是路过"的文字上。

配乐：轻钢琴+雨声，环境白噪音
运镜：推近→拉远→定格
时间密度：低密度（5-8字/秒）
```

### 《城市猎人》- 悬疑酷炫

```
赛博朋克电影风格，高对比度暗调，霓虹灯光，紧张压迫的情绪基调。

0-5秒：东京涩谷十字路口，暴雨中的霓虹灯在湿漉漉的地面上投下五颜六色的倒影。一个穿黑色风衣的男人撑着透明伞站在斑马线中央，镜头从脚底缓慢上摇至面部，他戴着夜视眼镜，眼神冷峻。

5-10秒：男人突然收起伞，雨水顺着他的风衣领口流下。他快步穿过马路，路边的监控摄像头红光闪烁同步跟进。远处传来直升机的声音，他回头看一眼，右手伸进风衣内袋。镜头快速切到他的特写：嘴角微微上扬。

10-15秒：男人走进一条狭窄的巷子，墙壁上贴满泛黄的海报和霓虹灯管。镜头跟随他进入一个转角，随后急停——面前是一扇带有指纹识别锁的铁门。男人把手按上去，门缓缓打开，门内透出蓝紫色的光。镜头推进至门内，然后黑屏。画面出现"未完待续"四个字。

配乐：紧张低频电子音效+直升机声
运镜：上摇→跟随→快切→特写→推进
时间密度：高密度（15-20字/秒）
```

### 《父亲的秘密》- 情感催泪

```
纪录片风格，低饱和度冷调，真实感，温馨治愈的情绪基调。

0-5秒：一个老旧的木质书架占据画面大部分，后面传来翻纸声。镜头缓慢向右移动，一只手布满皱纹的手从书架上取下一个相册。翻开相册，泛黄的照片上是年轻时的父亲抱着刚出生的孩子。

5-10秒：镜头慢慢拉远，父亲坐在老花镜后的眼睛微微泛红。他翻到下一页，是孩子小时候学走路的照片，他的手指轻轻抚摸照片上孩子的小脚丫。窗外是黄昏的光线，照在他花白的头发上。背景只有老式座钟的滴答声。

10-15秒：父亲站起身，拄着拐杖慢慢走到窗边。窗外夕阳西下，他的影子在地板上拉得很长。镜头从背后拍摄他的背影，窗台上摆着孩子小时候的照片和一辆旧的自行车模型。他轻声说："你说你想看看世界，爸爸等了你30年。"画面缓缓暗下去，留下窗边剪影。

配乐：轻柔钢琴+座钟滴答
运镜：移动→拉远→侧拍→俯拍→拉远
时间密度：低密度（5-8字/秒）
```
