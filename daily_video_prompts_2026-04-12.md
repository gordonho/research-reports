### 🌅 上午篇 - 《ICU里的创业者》

**时长**：先导片15秒 + 正片90秒

**主题**：真实创业故事 - 创业者在医院ICU边工作边创业的极限人生

**【先导片 - 15秒】**
```
【3秒开头】特写：心电图监护仪的波形
【7秒】快速剪辑：点滴瓶、创业者趴在病房小桌写代码、笔记本电脑屏幕上的代码、旁边是创业者打着点滴的手
【5秒钩子】字幕：「他在ICU里写出了融资百万的代码」
```

**【正片 - 90秒】**
```
【场景1：0-15秒】
画面：晨光中的ICU病房特写
解说：这是2024年最冷的冬天，张明确诊了突发性心肌炎。在ICU的病床上，他收到了投资机构的拒绝信。
同期声：监护仪的滴滴声

【场景2：15-40秒】
画面：特写创业者苍白的脸和坚定的眼神，他艰难地拿起笔记本电脑
解说：但他没有放弃。在抗生素和监护仪的陪伴下，他开始重新设计产品。
特写：代码行一行行增加

【场景3：40-60秒】
画面：护士帮忙调整点滴，投资人视频通话
解说：护士成了他的"技术顾问"，帮他递充电器。投资人被他打动，主动提出追加投资。
字幕：3个月后，产品上线，融资百万

【场景4：60-80秒】
画面：创业者康复出院，走出医院，阳光照在脸上
解说：有时候，最艰难的时刻，恰恰是最接近成功的时刻。
空镜：城市CBD的日出

【结尾：80-90秒】
黑底白字：「创业不是躺平，是躺着想站起来」
@你的账号名
```

**【五定法分析】**
- 定人：30-40岁男性科技创业者，曾有类似经历的受众
- 定景：ICU病房、医院、创业者的家
- 定调：坚韧、感人、励志
- 定音：监护仪音效 + 轻柔钢琴
- 定拍：特写为主，慢速推进，穿插空镜

**【热门元素】**：逆袭人生、健康与事业、真实故事改编

```

```
