# 工业富联（601138）深度分析：AI服务器绝对龙头万亿市值再出发

## 深度投资研究报告

---

## 一、公司概况：全球电子制造服务巨头

### 1.1 基本信息

工业富联全称为富士康工业互联网股份有限公司，股票代码601138.SH，于2018年6月在上海证券交易所上市。公司是全球领先的电子制造服务（EMS）企业，也是A股市场AI服务器领域的绝对龙头。

**注册资本**：176.98亿元

**控股股东**：鸿海精密工业股份有限公司（持股比例约53%）

**董事长**：刘扬伟

**总经理**：郑弘孟

**员工数量**：超过20万人

### 1.2 业务架构

工业富联的业务主要分为四大板块：

1. **云计算**：AI服务器、通用服务器、存储服务器、400G/800G高速交换机
2. **通信设备**：5G基站、路由器、交换机、光模块
3. **移动设备**：智能手机、平板电脑、可穿戴设备
4. **工业互联网**：精密模具、自动化设备、工业机器人

其中，**云计算业务**是公司增长最快的板块，也是市场最为关注的AI相关业务。

### 1.3 股权结构

公司控股股东为全球最大的电子代工厂商鸿海精密（富士康母公司），持股比例约为53%。这种股权结构使得工业富联能够充分利用鸿海在全球供应链中的优势地位，同时也为公司提供了稳定的技术和客户资源。

---

## 二、财务分析：营收突破9000亿，净利润创历史新高

### 2.1 2025年年度业绩

2025年3月10日，工业富联发布2025年年报，各项财务指标均创历史新高：

**营业收入**：9028.87亿元，同比增长48.22%

**归母净利润**：352.86亿元，同比增长51.99%

**扣非净利润**：343.72亿元，同比增长52.62%

**基本每股收益**：2.76元，同比增长51.64%

这是工业富联首次实现营收突破9000亿元大关，净利润也首次突破350亿元。值得注意的是，公司在2025年第四季度单季归母净利润达到128亿元左右，同比增长约60%，延续了强劲的增长势头。

### 2.2 收入结构分析

从收入结构来看，云计算业务成为公司增长的核心引擎：

**云计算业务**：预计占比超过45%，同比增长超过60%

**通信业务**：稳定增长，占比约25%

**移动设备业务**：占比约20%，略有下降

**其他业务**：占比约10%

最值得关注的是**AI服务器业务**，据公司披露，2025年AI服务器收入同比增长超过300%，这意味着AI服务器业务收入可能已经突破2000亿元，成为公司最核心的增长动力。

### 2.3 盈利能力分析

**毛利率**：约7.8%，同比提升0.3个百分点

**净利率**：约3.9%，同比提升0.2个百分点

**ROE**：约18%，同比提升3个百分点

尽管毛利率在制造业中处于中等水平，但公司通过规模效应和产品结构优化，实现了盈利能力的持续提升。净利率从2019年的约2.5%提升至2025年的约3.9%，体现了公司从传统代工向高端制造转型的成效。

### 2.4 现金流状况

**经营活动现金流**：约400亿元

**自由现金流**：约300亿元

**资产负债率**：约55%

公司现金流状况良好，经营活动现金流能够覆盖资本开支需求。较低的资产负债率为公司在AI算力基础设施持续投资提供了充足的财务弹性。

### 2.5 2026年业绩展望

根据市场一致预期，工业富联2026年：

**营业收入**：预计突破1.1万亿元，同比增长约20%

**归母净利润**：预计约420-450亿元，同比增长约20-25%

**EPS**：预计约2.89-3.10元

这一增长预期主要基于：

- AI服务器需求持续爆发
- 800G高速交换机放量
- GB200、GB300系列服务器订单落地
- 液冷技术应用提升产品价值量

## 三、业务分析：AI服务器绝对龙头

### 3.1 AI服务器业务：增长超300%的核心引擎

工业富联是全球最大的AI服务器制造商，也是英伟达最重要的合作伙伴之一。公司为英伟达代工生产GB200、HGX、H100等系列AI服务器，在AI算力供应链中占据不可替代的位置。

**业务亮点**：

1. **AI服务器收入同比增长超300%**：这是2025年公司最亮眼的数据，表明AI服务器业务已经进入高速增长期

2. **800G高速交换机量价齐升**：随着数据中心网络速率从400G向800G演进，公司高速交换机业务迎来爆发式增长

3. **液冷技术领先**：公司已实现液冷服务器量产，能够有效降低数据中心PUE，符合“双碳”政策要求

4. **PCB价值量提升**：AI服务器单台PCB价值量约5000元，是传统服务器的3倍以上，产品结构持续优化

### 3.2 市场份额与竞争地位

根据市场研究数据，工业富联在全球AI服务器市场的份额约为**25-30%**，是绝对的市场龙头。主要竞争对手包括：

- 纬创资通（Wistron）
- 广达电脑（Quanta）
- 仁宝电脑（Compal）
- 英业达（Inventec）

相比竞争对手，工业富联的优势在于：

1. **规模优势**：全球最大的电子制造产能，能够满足大客户批量订单需求

2. **技术优势**：在散热、结构设计、良率控制等领域积累深厚

3. **客户关系**：与英伟达、微软、谷歌、亚马逊等头部客户建立长期合作关系

4. **垂直整合**：依托鸿海集团的供应链资源，具备更强的成本控制能力

### 3.3 GB200/GB300系列服务器

2025年，英伟达发布GB200系列AI服务器，这是当前性能最强的AI算力基础设施。工业富联作为英伟达的主要代工伙伴，将在未来两年内持续受益于GB200系列服务器的放量。

**GB200 NVL72**：搭载72颗Blackwell GPU，适合大规模AI训练

**GB300系列**：预计2026年推出，性能进一步提升

市场预期，2026年GB200/GB300系列服务器将为工业富联贡献约**500-800亿元**的收入增量。

### 3.4 通信业务：5G与数据中心双轮驱动

工业富联的通信业务主要包括：

- **5G基站**：为华为、爱立信、诺基亚等设备商代工
- **数据中心交换机**：400G/800G高速交换机
- **光模块**：高速光模块产品

2025年，公司800G交换机已实现规模化出货，预计2026年800G交换机占比将进一步提升。作为数据中心网络升级的核心受益者，通信业务将保持稳定增长。

### 3.5 工业互联网业务：长期增长潜力

公司工业互联网业务主要面向制造业客户提供智能化改造解决方案。虽然目前营收占比较小，但这是公司向高端制造转型的战略方向。

**业务模式**：提供从硬件到软件的一站式智能制造解决方案

**目标客户**：汽车零部件、电子制造、医疗器械等行业

**发展潜力**：随着制造业数字化转型加速，工业互联网业务有望成为新的增长点

## 四、行业分析：AI算力需求持续爆发

### 4.1 全球AI服务器市场

根据市场研究机构预测，2026年全球AI服务器市场规模将超过**7100亿美元**，同比增长约61%。这一增长速度远超传统IT市场，反映了AI算力基础设施正处于新一轮投资周期。

**驱动因素**：

1. **大模型训练需求**：GPT-5、Claude 4等大模型持续迭代，对算力需求指数级增长

2. **推理需求爆发**：AI应用场景不断落地，从云端推理到边缘推理需求全面开花

3. **数据中心升级**：传统数据中心向AI数据中心转型，400G/800G网络加速普及

4. **自动驾驶**：L4级自动驾驶商业化加速，车载算力需求大幅增长

### 4.2 中国AI服务器市场

中国市场是工业富联最重要的收入来源之一。受益于：

- 国产大模型快速发展（豆包、文心一言、通义等）
- 运营商AI算力投资增加
- 互联网厂商AI基础设施布局
- 政策支持算力基础设施建设

预计2026年中国AI服务器市场规模将达到**2000亿元人民币**，同比增长约40%。

### 4.3 竞争格局

**全球AI服务器ODM市场**：

- 工业富联：约25-30%份额
- 纬创资通：约20%份额
- 广达电脑：约15%份额
- 其他：约40%份额

工业富联凭借规模优势和客户关系，稳居行业第一梯队。

### 4.4 产业链分析

**上游**：GPU（英伟达、AMD）、CPU（英特尔、AMD）、内存（三星、SK海力士）、PCB（鹏鼎控股、深南电路）

**中游**：服务器制造（工业富联、纬创、广达等）

**下游**：云服务商（微软、谷歌、亚马逊、腾讯、阿里巴巴等）、运营商、企业客户

工业富联处于产业链中游，兼具规模优势和客户资源优势，议价能力较强。

## 六、投资逻辑

### 6.1 核心投资逻辑

**1. AI算力超级周期受益者**

AI大模型竞赛进入白热化阶段，算力需求呈指数级增长。工业富联作为全球AI服务器龙头，是AI算力超级周期的核心受益者。2025年AI服务器收入增长超300%，2026年有望继续维持高速增长。

**2. 英伟达最重要合作伙伴**

工业富联是英伟达GB200系列服务器的主要代工厂商，随着Blackwell架构GPU全面放量，公司将持续受益于AI算力硬件需求爆发。

**3. 800G交换机第二增长曲线**

数据中心网络从400G向800G升级，工业富联800G交换机已实现规模化出货，产品价值量高于传统交换机，将成为重要的利润增长点。

**4. 万亿市值龙头地位稳固**

公司是A股市场首家万亿市值的AI服务器概念股，具有显著的规模优势和客户资源优势，行业地位难以撼动。

### 6.2 增长驱动因素

1. **AI服务器产能扩张**：持续投入扩产，满足客户订单需求
2. **产品结构优化**：AI服务器占比提升，毛利率持续改善
3. **新技术应用**：液冷服务器、800G交换机等高端产品放量
4. **海外布局**：在墨西哥、越南等地建厂，贴近北美客户

### 6.3 风险提示

1. **大客户依赖风险**：对英伟达、苹果等大客户依赖度较高
2. **行业竞争加剧**：台湾代工厂商竞争激烈
3. **宏观经济波动**：消费电子需求可能承压
4. **地缘政治风险**：中美科技竞争可能影响供应链
5. **估值回调风险**：短期涨幅较大，可能面临回调压力

## 七、深度讨论：工业富联的护城河

### 7.1 规模护城河

工业富联拥有全球最大的电子制造产能，年产值近万亿元。这种规模优势体现在：

- **成本优势**：规模化采购降低原材料成本
- **产能优势**：能够满足大客户批量订单需求
- **效率优势**：成熟的生产管理经验，良率高、成本低

在AI服务器这种高度标准化的制造领域，规模往往是最重要的竞争优势。

### 7.2 客户护城河

公司与英伟达、微软、谷歌、亚马逊、苹果等全球科技巨头建立了长期合作关系。这种客户护城河体现在：

- **信任壁垒**：头部客户不会轻易更换供应商
- **合作深度**：参与客户产品研发，而非简单代工
- **订单保障**：长期合作协议提供业绩确定性

### 7.3 技术护城河

工业富联在服务器制造领域积累了大量技术优势：

- **散热技术**：液冷服务器量产技术领先
- **PCB设计**：高密度PCB设计能力
- **测试验证**：完整的可靠性测试体系
- **供应链管理**：高效的全球供应链协同

### 7.4 产业链整合护城河

依托鸿海集团，工业富联具备独特的产业链整合能力：

- **上游原材料**：与供应商建立战略合作
- **生产设备**：自主研发自动化设备
- **下游客户**：深度参与客户产品规划
