## 一、公司概况

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 证券代码 | 002498.SZ |
| 证券简称 | 汉缆股份 |
| 上市时间 | 2010年 |
| 注册地址 | 山东省青岛市 |
| 主营业务 | 电线电缆研发、生产、销售 |
| 所属板块 | 电力设备-电网设备-线缆部件及其他 |
| 概念板块 | 特高压、高低压设备、风能、海上风电、海洋经济 |

### 1.2 主营业务

汉缆股份是中国领先的电线电缆制造商，主要产品包括：

- **电力电缆**：高压、超高压电力电缆
- **海底电缆**：海上风电场用电缆
- **电气装备用电缆**：工业设备配套电缆
- **裸导线**：输电线路用导线

公司可为算力数据中心提供相关产品，如电力电缆、氢能相关产品等。

**解读**：2025年公司营收和利润均实现增长，业绩稳健。

### 3.2 2025年三季度数据

| 指标 | 2025Q3 | 同比变化 |
|------|--------|----------|
| 营业总收入 | 71.76亿元 | +9.68% |
| 归母净利润 | 4.74亿元 | -18.55% |

**解读**：

- 应收账款体量较大（应收账款/利润达524.25%），需关注回款风险

## 四、核心竞争力分析

### 4.1 技术优势

- **超高压电缆技术**：国内少数具备500kV及以上超高压电缆生产能力的企业
- **海底电缆技术**：具备海上风电场用电缆研发和制造能力
- **研发投入**：持续投入研发，提升产品附加值

### 4.2 资质优势

- 拥有完整的电线电缆生产资质
- 通过国内外多项认证
- 参与国家重大工程项目

### 4.3 区位优势

- 总部位于青岛，拥有港口优势
- 临近东部沿海风电场
- 出口业务便利

## 五、行业分析

### 5.1 电力出海：2026年最确定主线

#### 5.1.1 美国超高压电网扩建

- 美国宣布大规模超高压电网扩建计划
- 需要大量高压、超高压电缆
- 中国电缆企业具备成本和技术优势

#### 5.1.2 全球电网设备需求

- 新兴市场电网建设加速
- 发达国家电网老化改造
- 可再生能源并网带动电缆需求

### 5.2 国内市场机遇

- **特高压建设**：国内特高压项目持续推进
- **海上风电**：海上风电场建设带动海底电缆需求
- **数据中心**：AI数据中心建设带动电力电缆需求

### 5.3 行业竞争格局

|------|----------|

## 九、结论

汉缆股份作为国内电线电缆龙头企业，在超高压电缆和海底电缆领域具有技术优势。近期股价大幅上涨主要受益于"电力出海"概念和美国超高压电网扩建预期。

**风险提示**：公司当前估值较高，基本面有走弱迹象，建议谨慎对待短期涨幅，等待更好的介入时机。中长期来看，电力出海是确定性主线，可持续关注公司海外订单落地情况。
