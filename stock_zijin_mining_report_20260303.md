## 二、公司概况

### 2.1 基本信息

紫金矿业集团股份有限公司（简称"紫金矿业"）成立于1986年，总部位于福建省上杭县。公司于2004年在上海证券交易所上市，股票代码601899。

**主营业务：**
- 矿产资源勘查与开发
- 黄金、铜、锌、铁等金属矿产
- 锂矿（新能源布局）

**全球地位：**
- 矿产金产量：全球前六
- 矿产铜产量：全球前四
- 市值：全球矿业公司第二

### 2.2 发展历程

```
1986年：紫金矿业成立（上杭紫金山金矿）
2004年：A股上市
2015年：进入海外并购阶段
2020年：铜矿产能快速扩张
2025年：市值突破万亿，成为"金王"
2026年：收购联合黄金，冲击全球金矿前三
```

## 三、投资逻辑：为什么看好紫金矿业？

### 3.1 逻辑一：金铜价格共振，业绩弹性巨大

**黄金：避险需求+美元走弱**

2025年以来，全球宏观环境呈现以下特点：
- 地缘政治冲突加剧（俄乌、中东）
- 美联储降息预期持续
- 美元指数走弱
- 全球央行购金热潮

这些因素共同推动金价持续上涨，沪金均价同比增长38.6%。

**铜：AI算力+新能源需求爆发**

铜是AI数据中心、电力传输、新能源汽车的核心原材料：
- AI服务器铜需求是传统服务器的5-8倍
- 全球新能源装机量持续增长
- 铜矿供给偏紧，TC/RC费用处于低位

**紫金矿业受益逻辑：**
- 矿产金毛利率46%（2024年）
- 矿产铜毛利率66%（2024年）
- 量价齐升带来业绩高弹性

### 3.2 逻辑二：产能持续释放，成长性凸显

3. **并购联合黄金** - 新增金矿产能
4. **Kurmuk金矿** - 2026年下半年投产
